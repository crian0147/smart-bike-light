## 二、电源系统

### 2.1 电池配置

```
方案：2 节 18650 并联 → 单节电压 3.0-4.2V

选择并联而非串联的理由：
  ✅ 直接 3.7V 标称，降压到 3.3V 效率高（LDO 损耗小）
  ✅ 充电管理简单，两路 TP4056 各管一节
  ✅ 容量翻倍（3000mAh × 2 = 6000mAh）
  ✅ 安全性高，单节故障不影响另一节

如果不并联要串联 (7.4V)：
  ❌ 需要降压比更大，LDO 发热严重
  ❌ 需要 2S BMS 均衡充电，复杂度高
  ❌ WS2812B 要额外 5V 升压或 buck
```

### 2.2 电源轨设计

```
电池 (3.0-4.2V)
    │
    ├──→ TP4056 充电（USB-C 5V 输入，1A 充电电流）
    │         │
    │    ┌────┴────┐
    │    │ DW01A   │    ← 锂电池保护 IC（过充/过放/过流）
    │    │ FS8205A │    ← 保护 MOSFET
    │    └────┬────┘
    │         │ 电池输出 3.0-4.2V
    │
    ├──→ [V_BAT] 3.0-4.2V
    │         │
    │         ├──→ AMS1117-3.3 ──→ [V_3V3] 供 ESP32/IMU
    │         │         │
    │         │         └──→ 滤波：100µF + 10µF + 0.1µF
    │         │
    │         └──→ WS2812B LED 数据电压 3.3V (ESP32 输出)
    │                   │
    │                   └──→ LED 供电需要 ≥ 3.7V，直连 V_BAT
    │                        但 WS2821B 在 3.0V 时红色 LED 可能偏暗
    │                        ★ 建议加一个升压到 5V 给 LED 供电
    │
    └──→ [V_LED] 方案：
              电池 3.0-4.2V → MT3608 升压 → 5V → WS2812B VCC
              MT3608 效率 ~93%，可调输出，便宜（¥1）
```

### 2.3 修正后的电源架构

```
USB-C (5V)
    │
    ├──→ TP4056 #1 ──→ 电池 #1 (18650) ──┐
    │                                      ├── 并联 ──→ [V_BAT] 3.0-4.2V
    └──→ TP4056 #2 ──→ 电池 #2 (18650) ──┘
                                           │
                                    ┌──────┼──────┐
                                    │      │      │
                              AMS1117-3.3  │   MT3608
                              [V_3V3]      │   升压→5V
                                    │      │      │
                                    │   V_BAT   [V_5V_LED]
                                    │  (直供)     │
                           ESP32-S3    │   WS2812B
                           ICM-20948    │
                           霍尔传感器    │
                           光敏电阻      │
                                       (LED 5V 供电)
```

### 2.4 电源参数

| 电源轨 | 电压 | 负载 | 最大电流 |
|--------|------|------|---------|
| V_BAT | 3.0-4.2V | — | — |
| V_3V3 | 3.3V | ESP32 + IMU + 霍尔 + 光敏 | ~50mA |
| V_5V_LED | 5.0V | WS2812B (60颗全亮白) | ~3.6A (峰值) |
| V_5V_LED | 5.0V | WS2812B (60颗红色80%) | ~1.4A (典型) |

> ⚠️ WS2812B 全亮白色时电流很大，建议限制单颗最大亮度到 80%

### 2.5 电量监测

```
V_BAT ──→ 分压电阻 ──→ ESP32 ADC1_CH3 (GPIO3)

分压计算：
  V_bat × R2 / (R1 + R2) ≤ 3.3V
  4.2V × R2 / (R1 + R2) ≤ 3.3V
  
  R1 = 10KΩ, R2 = 10KΩ
  → ADC读数 = V_bat × 10 / (10+10) = V_bat / 2
  → 4.2V → ADC = 2.1V ✓
  → 3.0V → ADC = 1.5V ✓
  
  电量百分比估算：
    4.2V = 100%
    4.0V = ~80%
    3.7V = ~50%
    3.3V = ~10%
    3.0V = 0% (过放保护触发)
```

## 三、ESP32-S3 核心电路

### 3.1 最小系统

```
ESP32-S3-WROOM-1 (N8R8)
┌─────────────────────────────────────────┐
│                                         │
│  VDD  ────── 3.3V                       │
│  GND  ────── GND                        │
│                                         │
│  EN   ───┬── 10KΩ ─── 3.3V (上拉)       │
│          └── 100nF ── GND (复位电容)     │
│                                         │
│  IO0  ─── 10KΩ ─── 3.3V (启动模式选择)   │
│          运行模式 IO0=HIGH               │
│          下载模式 IO0=LOW                │
│                                         │
│  TXD0/RXD0 ─── USB-TTL (调试串口)        │
│  (用于开发调试，量产可省略)               │
│                                         │
│  strapping pins 按官方推荐接法：          │
│  IO45 ─── 10KΩ ─── 3.3V                │
│  IO46 ─── 10KΩ ─── 3.3V                │
│  IO0  ─── 10KΩ ─── 3.3V                │
│                                         │
│  GND  ─── 靠近 VDD 处放 0.1µF 去耦电容   │
│                                         │
│  天线  ─── PCB 天线区域不铺铜            │
│  (使用 WROOM 内置 PCB 天线版本)          │
│                                         │

```

### 3.2 引脚总分配表

| GPIO | 方向 | 功能 | 外设 | 备注 |
|------|------|------|------|------|
| 1 | ADC_IN | 光敏电阻 | ADC1_CH0 | 模拟输入 |
| 3 | ADC_IN | 电池电压 | ADC1_CH3 | 模拟输入 |
| 4 | INPUT | 霍尔传感器信号 | GPIO | 外部中断，上升沿 |
| 5 | INPUT | 防盗解锁按钮 | GPIO | 上拉，低有效 |
| 8 | OUTPUT | I²C SDA | I2C0_SDA | 接 ICM-20948 |
| 9 | OUTPUT | I²C SCL | I2C0_SCL | 接 ICM-20948 |
| 17 | TX | UART2 TX → 雷达 RX | UART2 | 可选，雷达也支持 GPIO 控制 |
| 18 | RX | UART2 RX ← 雷达 TX | UART2 | 可选 |
| 43 | TX | UART1 TX → GPS RX | UART1 | NMEA 数据接收 |
| 44 | RX | UART1 RX ← GPS TX | UART1 | NMEA 数据接收 |
| 46 | OUTPUT | GPS 电源控制 | GPIO | HIGH=供电, LOW=断电省电 |
| 47 | OUTPUT | 雷达电源控制 | GPIO | HIGH=供电, LOW=断电 |
| 48 | OUTPUT | WS2812B 数据 | RMT | LED 灯带信号线 |

> ⚠️ ESP32-S3 的 strapping pin（IO0/IO3/IO45/IO46）在启动时有特定电平要求
> - IO0 已用作 ADC（ADC1_CH3），启动时通过 10K 上拉确保 HIGH（正常启动）
> - IO45、IO46 上拉到 3.3V
> - 如需进入下载模式，需要短暂拉低 IO0

### 3.3 防盗按钮电路

```
V_3V3 ──┬── 10KΩ ──┬── GPIO5
        │           │
       [BTN]      100nF
        │        ──┤├── GND
        │
       GND

按下按钮 → GPIO5 = LOW → 触发中断
松开按钮 → GPIO5 = HIGH (上拉)
100nF 电容：软件消抖滤波
长按 5s → 解除防盗模式
```

## 四、传感器电路

### 4.1 ICM-20948（IMU）

```
ESP32-S3                    ICM-20948
                            
GPIO8 (SDA) ──────── SDA
GPIO9 (SCL) ──────── SCL
V_3V3      ──────── VDD    (3.3V)
V_3V3      ──────── VDDIO  (3.3V I/O 电平)
GND        ──────── GND

AD0 (地址选择) ──── GND    (I²C 地址 0x68)
                         或 3.3V → 地址 0x69

I²C 上拉（每个线路上）：
  V_3V3 ── 4.7KΩ ── SDA
  V_3V3 ── 4.7KΩ ── SCL

去耦电容：
  VDD 旁：0.1µF + 10µF
  VDDIO 旁：0.1µF
```

### 4.2 霍尔传感器 A3144

```
            V_3V3
             │
        ┌────┴────┐
        │ A3144   │
        │         │
VDD ────┤1       3├──── GND
        │         │
        │    OUT  ├────┬──── GPIO4 (ESP32)
        └─────────┘    │
                      10KΩ (上拉，可选)
                       │
                      V_3V3

安装方式：
  - A3144 固定在前叉/车架上（贴近但不接触轮圈）
  - 钕铁硼磁铁（10×3mm）贴在轮圈辐条上
  - 转一圈产生一个脉冲
  - 车轮周长 / 脉冲间隔 = 速度
```

**速度计算：**
```
假设 700C 轮胎周长 ≈ 2.135m

脉冲间隔 T (秒) = 两脉冲之间时间
速度 = 周长 / T = 2.135 / T (m/s)
速度 (km/h) = 2.135 / T × 3.6 = 7.686 / T

采样策略：
  - GPIO4 上升沿触发外部中断
  - 中断里记录时间戳
  - 定时器每 100ms 计算一次速度
  - 如果 > 1s 没收到脉冲 → 速度 = 0
```

### 4.3 HLK-LD2410 微波雷达

```
供电控制方式（推荐，可省电）：

ESP32 GPIO47 (PWR_CTRL)
    │
    ├──→ N-MOSFET (S8050 / AO3400)
    │         │
    │    V_BAT ──→ MOSFET Drain
    │         MOSFET Source ──→ LD2410 VCC
    │
    └── HIGH: 雷达上电, LOW: 雷达断电

或者简单直连（不省电）：
  V_3V3 ──────── LD2410 VCC
  GND   ──────── LD2410 GND

通信方式（二选一）：

方式 A — UART（推荐，数据更丰富）：
  ESP32 GPIO18 (RX) ── LD2410 TX
  ESP32 GPIO17 (TX) ── LD2410 RX
  → 可获取：目标距离、移动/静止状态、信号强度

方式 B — GPIO 输出（简单）：
  LD2410 OUT ── ESP32 GPIO（任意）
  → 只能获取：有/无目标（开关量）
  → 不推荐，丢失距离信息
```

**推荐方式 A（UART），可以获取目标距离，做渐变灯效。**

### 4.4 ATGM336H GPS 模块

```
供电控制：
  ESP32 GPIO46 (GPS_PWR) ──→ N-MOSFET ──→ ATGM336H VCC
  （停车时可断电省 25mA）

通信：
  ESP32 GPIO43 (TX) ──→ ATGM336H RX  (模块波特率默认 9600)
  ESP32 GPIO44 (RX) ←── ATGM336H TX

供电：
  ATGM336H VCC ──→ V_5V (3.3V 也可，但 5V 更稳)
  ATGM336H GND ──→ GND

PPS 输出（可选）：
  ATGM336H PPS ──→ ESP32 GPIO（计时用，可选）
```

**注意：** GPS 模块天线朝上安装，金属遮挡会严重影响信号。尾灯安装位置（座管下方）需确认天线有足够天空视野。

### 4.5 光敏电阻 GL5528

```
V_3V3 ──┬── GL5528 (光敏电阻)
        │         │
        │        10KΩ
        │         │
        └─────┬───┘
              │
        ┌─────┴─────┐
        │  100nF    │
        │  ──┤├── GND│ ← 滤波
        └─────┬─────┘
              │
         ESP32 GPIO1 (ADC1_CH0)

ADC 读数范围：
  暗环境（夜晚）：~2500-3500 (12-bit ADC, 0-4095)
  室内正常：~1500-2500
  户外阴天：~500-1500
  户外晴天：~100-500
  直射阳光：~50-100

阈值设定（可调）：
  < 800 → 夜间模式（LED 亮度 40%）
  > 1500 → 日间模式（LED 亮度 100%）
  中间 → 渐变过渡
```

## 五、WS2812B LED 电路

### 5.1 驱动方案

```
ESP32-S3 GPIO48 (RMT) ───── 470Ω ───── WS2812B DIN
                                       │
MT3608 5V ───────┬─────────────── WS2812B VCC
                 │                    │
               470µF              每 60 颗:
              (电解)               C1: 0.1µF (去耦)
                 │               │
                GND ──────── WS2812B GND
```

### 5.2 LED 数量与布局

```
推荐使用 60 颗 LED，灯带裁剪成特定形状：

外壳形状：圆角矩形，约 80mm × 30mm

LED 布局 (8行 × 7-8列 ≈ 60颗):

  ○ ○ ○ ○ ○ ○ ○ ○    ← 第1行 (8颗)
  ○ ○ ○ ○ ○ ○ ○ ○    ← 第2行 (8颗)
  ○ ○ ○ ○ ○ ○ ○ ○    ← 第3行 (8颗)
  ○ ○ ○ ○ ○ ○ ○ ○    ← 第4行 (8颗)
  ○ ○ ○ ○ ○ ○ ○ ○    ← 第5行 (8颗)
  ○ ○ ○ ○ ○ ○ ○ ○    ← 第6行 (8颗)
  ○ ○ ○ ○ ○ ○ ○      ← 第7行 (7颗)
  ○ ○ ○ ○ ○ ○ ○      ← 第8行 (7颗)
  ─────────────────
  左转区(18) | 主体(24) | 右转区(18)

数据线走向：蛇形走线（每行末尾折返）
  行1: 左→右
  行2: 右→左
  行3: 左→右
  ...
```

### 5.3 电流计算

```
单颗 WS2812B 电流：
  红色 100%: ~20mA
  绿色 100%: ~20mA  
  蓝色 100%: ~20mA
  白色 100%: ~60mA

60 颗全部红色 80%: 60 × 20 × 0.8 = 960mA ≈ 1A
60 颗全部白色 100%: 60 × 60 = 3600mA = 3.6A ← 峰值

实际使用中：
  刹车全红闪烁(80%): ~1A
  匀速红色常亮(50%): ~600mA
  来车提醒闪烁: ~800mA
  停车慢闪(30%): ~360mA

MT3608 升压模块规格：
  输入：2V-24V
  输出：5V-28V (可调)
  最大电流：2A
  效率：~93%

⚠️ 如果 60 颗全白需要 3.6A，MT3608 的 2A 不够
建议：
  方案 A：限制白色使用，实际骑行只用红/橙/绿，峰值不超过 1.5A ✓
  方案 B：换用更大电流的升压模块（如 SY8120，3A，¥3）
  ★ 推荐方案 A，尾灯法规要求红色为主
```

## 八、注意事项

1. **ESP32-S3 IO0 兼容性**：IO0 用作 ADC 采集电池电压，同时也是 strapping pin。启动时需要上拉到 3.3V（下载模式需短接 GND），正常运行时保持 HIGH。TP4056 充电时电池电压可能达到 4.2V，分压后 ADC 读数仍在安全范围。

2. **WS2812B 数据线**：GPIO48 → 470Ω → DIN，这颗电阻很重要，防止信号反射导致灯效错乱。

3. **I²C 上拉**：ICM-20948 模块有些已自带上拉，买回来先用万用表量一下 SDA/SCL 到 VDD 是否有 4.7KΩ 电阻。没有的话需要外接。

4. **GPS 天线方向**：ATGM336H 模块上方的陶瓷天线必须朝上，不能被金属遮挡。安装尾灯时考虑天线朝向。

5. **雷达安装**：LD2410 微波雷达通过检测多普勒频移判断物体移动，安装时传感器面朝车后方，检测范围覆盖后方车道。

6. **防水**：Phase 1 面包板阶段不考虑防水，Phase 4 正式外壳阶段再做。面包板阶段用热缩管保护接头即可。
