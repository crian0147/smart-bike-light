## ESP32 LED 控制项目

### 项目概况
- **ESP32S3_light_tool** — Flutter 上位机，通过 TCP 控制 ESP32-S3 的 LED 亮度
- **ESP32S3_LED_Ctrl** — ESP32 固件端，PWM + MCP4725 DAC 控制外部 LED（tps92641）
- GitHub: crian0147/ESP32S3_light_tool, crian0147/ESP32S3_LED_Ctrl
- 通信协议：帧头 `55 AA` + length + cmd + data + CRC8(多项式0x31) + 帧尾 `AA 55`

### 发现并修复的 Bug（Flutter 上位机 — 分支 fix/protocol-bugs）
1. **CRC8 算法错误**: Dart int 不会自动截断，移位后需 `& 0xFF`
2. **parseResponse 帧长度校验差 1**: `3+length+2` → `2+length+2`
3. **多余 import**: `dart:typed_data` 未使用
4. **测试数据写错**: 3000 写成 0x0B58（2904），应为 0x0BB8
5. **brightness_slider 两处 unnecessary_cast**
6. **widget_test.dart**: 残留 Counter 模板测试

### 修复的 Bug（Python 脚本 — 分支 fix/python-protocol）
1. **length 字段漏算 CRC**: `1+len(data)` → `1+1+len(data)+1`
2. **响应解析逻辑混乱**: 重写 _parse_response，正确处理 CRC 错误和正常响应
3. **set_mode 缺少 HYBRID 模式**: 新增 mode 2 支持

### 验证结果
- Flutter: `flutter test` 15/15 passed, `flutter analyze` 0 error 0 warning
- Python 帧输出与 Flutter 完全一致: `55 AA 05 B1 0B B8 C7 AA 55`
