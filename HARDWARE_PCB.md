# PCB 设计指南

> 智能单车尾灯 — PCB Layout 指南
> 日期：2026-04-02

> 本文档面向 PCB 设计阶段（Phase 4 之前为面包板/洞洞板验证，本文件为后期正式 PCB 化准备）

---

## 一、PCB 概述

### 1.1 设计目标

```
板型：圆角矩形，匹配尾灯外壳
尺寸：约 85mm × 35mm × 双层板
工艺：2层 FR4，厚度 1.6mm
表面处理：HASL（喷锡）或 ENIG（沉金，贵但更好焊）
最小线宽/线距：0.2mm / 0.2mm
最小过孔：0.4mm（推荐 0.6mm）
```

### 1.2 层叠结构

```
Top Layer    ── 信号走线 + 元件 + WS2812B 灯带焊盘
Bottom Layer ── 信号走线 + 地平面铺铜
                 ↓
          FR4 1.6mm
                 ↑
Top GND      ── 局部地平面（大电流回路）
Bottom GND   ── 完整地平面铺铜
```

### 2.2 模块位置原则

| 模块 | 放置区域 | 原因 |
|------|---------|------|
| WS2812B 灯带 | 板面中央偏上 | 发光面朝外，最大可视面积 |
| ESP32-S3 | 板面下方中央 | 方便走线到所有外设 |
| ICM-20948 | ESP32 旁边 | I²C 线要短 |
| AMS1117-3.3 | V_BAT 附近 | 减少压降 |
| MT3608 升压 | LED 区域附近 | 5V 走线到 LED 要短且粗 |
| TP4056 × 2 | 板边 USB-C 口旁 | USB-C 信号走线短 |
| LD2410 雷达 | 板边朝后方 | 微波朝车后方检测 |
| 光敏电阻 | 板面朝上（或外壳侧面开孔） | 感测环境光 |
| ATGM336H GPS | 板边朝上方 | 天线需朝天空 |
| 防盗按钮 | 外壳上独立开孔位置 | 用户可触及 |

## 三、关键走线规则

### 3.1 电源走线

```
V_BAT (大电流路径):
  电池 → TP4056 → 主电源总线
  线宽: ≥ 1.0mm (2-3A 峰值考虑)
  加粗铺铜处理

V_5V_LED (最大电流):
  MT3608 输出 → WS2812B VCC
  线宽: ≥ 1.5mm
  沿路每颗 LED 旁放 0.1µF 去耦
  总线入口放 470µF 电解电容 + 100µF 瓷片

V_3V3:
  AMS1117 输出 → ESP32 + 传感器
  线宽: ≥ 0.5mm
  ESP32 旁: 10µF + 0.1µF
  ICM-20948 旁: 10µF + 0.1µF

GND:
  Bottom Layer 完整铺铜作为地平面
  Top Layer 大面积铺铜连通 Bottom GND（多个过孔）
  大电流 GND 回路要短且粗
```

### 3.2 信号走线

```
I²C (SDA/SCL):
  线宽: 0.2mm
  平行走线，间距 ≤ 0.5mm
  远离电源线和高速信号
  上下拉电阻靠近 ICM-20948 放置

WS2812B 数据线:
  线宽: 0.3mm
  尽量走直线，避免 90° 拐角（用 45°）
  不要穿过 LED 灯带下方（避免噪声耦合）
  ESP32 GPIO48 → 470Ω → 第一颗 LED DIN
  LED 之间 DOUT→DIN 间距匹配灯带节距

UART (GPS, 雷达):
  线宽: 0.2mm
  交叉时走 Bottom Layer
  加 TVS 管防护（户外 ESD 风险）

霍尔传感器:
  GPIO4 走线尽量短
  远离 WS2812B（LED 开关噪声可能干扰中断）
  如果交叉，GND 包围隔离
```

### 3.3 天线区域

```
ESP32-S3-WROOM 内置 PCB 天线：
  模块顶部 12mm × 15mm 区域 → 不铺铜
  不走信号线
  不放金属元件
  下方 GND 平面也需要相应挖空（参考模块 datasheet）
  
GPS 天线（ATGM336H 陶瓷天线）：
  下方不铺铜
  远离 WS2812B（LED 开关噪声）
  尽量靠近板边
```

## 四、外壳 3D 设计规格

### 4.1 外形尺寸

```
整体外形：圆角矩形
  外壳外部: 90mm × 40mm × 25mm (长 × 宽 × 高)
  壁厚: 2mm
  内部空间: 86mm × 36mm × 21mm
  
PCB 板: 85mm × 35mm × 1.6mm
固定方式: 4 × M2 螺丝柱
```

### 4.2 结构设计

```
两段式设计：

上半壳（灯罩）：
  - 材质：透明 PETG 或半透明红色 PETG
  - 厚度: 1.5mm（透光率好）
  - 紫外线抗氧化处理
  - 内表面做磨砂/扩散处理（柔光效果）
  - 完全密封

下半壳（底座）：
  - 材质：PETG 或 ABS（黑色）
  - 壁厚: 2mm
  - USB-C 充电口开孔（带硅胶塞防水）
  - 光敏电阻开孔（朝上）
  - 防盗按钮开孔（侧面）
  - 雷达微波透过区域（塑料可透微波，无需开孔）
  - GPS 天线区域不加盖金属

密封方式：
  - 上下壳之间: 硅胶密封圈 (2mm × 1mm 截面)
  - USB-C 口: 硅胶防水塞
  - 螺丝孔: 硅胶垫圈

安装方式（座管夹具）：
  - 底部带卡扣或抱夹，适配 25.4/27.2/31.6mm 座管
  - 或通用 3D 打印夹具（通过 M5 螺丝固定在 PCB 底座）
```

### 4.3 防水等级目标

```
Phase 1 (面包板): 不防水，室内测试
Phase 2 (3D打印壳): IP54（防泼水）
Phase 3 (正式壳): IP65（防喷水）
```

### 4.4 散热

```
WS2812B 是主要发热源：
  全亮白 60颗: ~216W (不可能长时间)
  实际刹车闪烁: ~5W (瞬态)
  匀速骑行: ~2W

散热方案：
  - PCB Bottom GND 铺铜作为散热面
  - 底壳金属化或加铝箔散热片（热传导到外壳）
  - 外壳侧面加散热条纹（自然对流）
  - 正常骑行风冷足以散热，停车时灯效自动降低亮度
```

## 五、连接器规格

### 5.1 外部连接

```
1. USB-C 充电口:
   - 型号: Type-C 母座 16Pin (沉板/直插)
   - 位置: 底壳侧面
   - 仅使用 VBUS(5V) 和 GND，不实现 USB 数据

2. 霍尔传感器外接线:
   - 方式: JST-PH 3Pin 或直接飞线
   - 线缆: 3芯硅胶线（VCC/GND/Signal）
   - 长度: 约 30cm（车架到轮圈距离）
   - 接头做防水处理（热缩管）

3. GPS 模块:
   - 方案 A: 直接焊接在 PCB 上（天线朝上）
   - 方案 B: 通过 4Pin 排针连接（便于调整天线方向）
   - 推荐: 方案 A，减少连接器故障点

4. 座管夹具:
   - PCB 底部预留 2 × M5 安装孔
   - 孔距匹配 3D 打印夹具
```
